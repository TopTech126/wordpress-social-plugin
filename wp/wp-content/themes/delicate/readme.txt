Installation

1. Primary: Login to your wp-admin area and go to Appearance -> Themes. Select Install tab and click on Upload link. Select theme .zip and ckick on Install now button. If you have any errors, use alternate method.
2. Alternate: Unzip the template file (Delicate.zip) that you have downloaded. Via FTP, upload the whole folder (delicate) to your server and place it in the /wp-content/themes/ folder. Do not change directory name. The template files should thus be here now: /wp-content/themes/delicate/index.php (for example).
3. Log into your WP admin panel and click on the Design tab. Now click on the Delicate theme to activate it.
4. Complete all of the required inputs on the Delicate Options page (in the WP admin panel) and click "Save Changes".
5. Read our Knowlegebase http://support.nattywp.com


How to Add Slideshow Image (delicate v.3.5)

Our slideshow works on a page-based functionality. While working with Slideshow consider pages as slides. 
To add more slides you should login to your wp-admin area and go to Pages -> Add New. Enter page title and content, then on the right sidebar you can find Featured Image. Click "Set featured image" link and attach proper image you'd like to use for slideshow (note that the image dimensions should not be less than 970x225 pixels)
Upload image and click "Use as Featured" link.

How to upload Header?

You can open Custom Header section. Go to Appearance -> Header and use Upload Image tool.
